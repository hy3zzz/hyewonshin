Export for GitHub Pages

GitHub Pages는 방문자 브라우저마다 localStorage가 달라서, 에디터 변경사항이 자동으로 배포되지 않아요.
아래 단계로 index.html에 콘텐츠를 직접 반영한 뒤 push하세요.

  1. 아래 코드를 전체 복사하세요 (Copy 버튼)
  2. index.html을 에디터(VSCode 등)로 열고
  3. const SITE_DATA = { 부터 닫는 }; 까지 선택해서 붙여넣기
  4. 저장 후 GitHub에 push → 배포 완료
